# Personal Cyber Security Page

This is a one-page personal site ready for GitHub Pages.

## 1) Update your links and email
Open `index.html` and update these values:
- `https://www.linkedin.com/` -> your real LinkedIn URL
- `https://github.com/` -> your real GitHub URL
- `your.email@example.com` -> your real email
- `https://YOUR_USERNAME.github.io/YOUR_REPO/` -> your final website URL

## 2) Publish on GitHub
If you already created an empty repo, run:

```bash
git init
git add .
git commit -m "Add personal cybersecurity landing page"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

## 3) Enable GitHub Pages
- Go to your repo on GitHub.
- Open Settings > Pages.
- Source: Deploy from a branch.
- Branch: `main` and folder `/ (root)`.
- Save.

Your site URL will be:
`https://YOUR_USERNAME.github.io/YOUR_REPO/`

## 4) Add it to LinkedIn
- Open your LinkedIn profile.
- In the Featured section, click `+`.
- Choose `Add a link`.
- Paste your GitHub Pages URL.
- Title suggestion: `Cyber Security Portfolio - Mohammed Al-Rasheed`.

## 5) Optional text for LinkedIn post
"I launched my cyber security portfolio page to showcase my certifications, skills, and current focus areas in ethical hacking and red team learning."
